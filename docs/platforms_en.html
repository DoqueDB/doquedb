System Requirements

[Go back to the top page]

You can run DoqueDB on the following environments:

Items Conditions
OS Linux (64bit)
* The operation checks have been done on the following environment.
  • RedHat Enterprise Linux 7 (gcc 4.8 version)
  • CentOS 7 (gcc 4.8 version)
  • CentOS Stream 9 (gcc 11.4 version)
Systems Architecture x86_64
File Systems Journaling file systems such as ext4, XFS
RAM 2GB or more
* The required RAM size varies depending on the data size.
Disk Size 1GB or more
* The required disk size varies depending on the data size.

Attention

The network connections from/to DoqueDB are not encrypted.
You must recognize this limitation and risks of sniffing attacks whenever you use DoqueDB.
